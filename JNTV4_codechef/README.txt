Implementación del problema 2.3-7 del libro
'Introduction to algorithms' y JNTV4 de
CodeChef.

Este algoritmo es una adaptación del merge
sort que determina si, dado un conjunto
S de n enteros y otro entero x, existen dos
elementos en S cuya suma sea exactamente x.

El programa espera un input tal como se
describe en la página correspondiente de
CodeChef


Instrucciones:
=============

1.) Compilar y correr:
--------------------------

cd src

make

cd ../bin/

./adp_merge


2.) Limpiar:
----------------

cd src

make clean
